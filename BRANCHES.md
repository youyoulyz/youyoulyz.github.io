# 分支说明

## main 分支
- 用于 GitHub Pages 托管的网站
- 部署地址：https://youyoulyz.github.io
- 此分支包含用于 GitHub Pages 的配置

## domestic 分支
- 用于阿里云服务器部署
- 部署地址：https://youyou.tech
- 此分支包含国内服务器的特殊配置，如备案号、HTTPS 设置等

## 修改指南
当对网站进行修改时，请确保同时更新两个分支：
1. 在 `main` 分支进行修改并提交
2. 在 `domestic` 分支进行修改并提交
3. 两个分支的修改应保持同步，除非有特定环境差异
